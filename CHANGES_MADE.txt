## MyQuestionsScreen.tsx
### Date: March 10, 2026
### Changes: New Screen Implementation

#### 1. Screen Purpose
- Display and manage student questions submitted to tutors
- Filter questions by status (All, Pending, Answered, Closed)
- Search questions by subject or content
- View question details with tutor, class, and assignment information

#### 2. API Integration
- **Endpoint**: GET /api/v1/student/questions
- **Service Method**: studentService.getQuestions()
- **Query Key**: ['studentQuestions']
- **Response**: Paginated list of questions with tutor, assignment, class, and attachments data

#### 3. UI Components

**Header Section:**
- Title: "My Questions"
- Subtitle: "View and manage your questions to tutors"
- Back navigation button

**Search Bar:**
- Magnifying glass icon on left
- Placeholder: "Search questions by subject or content..."
- Clear button (X icon) when text entered
- Real-time filtering

**Filter Tabs (Horizontal Scrollable):**
- 4 tabs: All Questions, Pending, Answered, Closed
- Each tab shows: Icon + Label + Count Badge
- Active tab: Green background, white text
- Inactive tab: White background, gray text
- Compact pill-shaped design
- Horizontal scroll for overflow

**Question Cards:**
- Header: Message icon + Subject + Status badge (color-coded)
- Question text (2 lines with ellipsis)
- Meta info: Class name + Assignment title (with icons)
- Footer: Tutor name + Attachment count + Created date
- White background with shadow and border

**Empty State:**
- Large message-circle icon
- "No questions found." text
- Centered display

#### 4. Features Implemented
- Pull-to-refresh functionality
- Status filtering (all/pending/answered/closed)
- Search by subject or question content
- Count badges update dynamically
- Horizontal scrolling tabs
- Responsive card layout
- Loading and error states

#### 5. Navigation Integration
- Added to StudentStackParamList as 'MyQuestions'

- Uses useNavigation hook for navigation

#### 6. Styling Details
- Compact pill-shaped filter tabs

#### 7. Technical Implementation
